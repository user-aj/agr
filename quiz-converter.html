How to add to agriedu4u.com (WordPress)

1Upload the HTML file via FTP or File Manager to your server (e.g. /public_html/quiz/rice-module-1.html)
2Or paste the file contents into a WordPress page using an HTML block (Gutenberg) or a plugin like WPCode
3Go to Google Search Console → URL Inspection → Request Indexing for the page
4Test the schema at search.google.com/test/rich-results
